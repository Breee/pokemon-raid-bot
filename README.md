# Pokemon-raid-bot
A discord Bot for Pokemon Go raid coordination.
Currently the Bot can only create Polls.

# Setup

#### 1. Install the discord python package usign pip3: 
```
pip3 install discord.py
```
#### 2. Adjust config.ini
```
token=<token>
```
todo. 

#### 3. Run the bot

```
python3 start_bot.py
```
